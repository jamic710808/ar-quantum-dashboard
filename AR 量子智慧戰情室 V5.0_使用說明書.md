## 1. 系統簡介

### 1.1 系統定位

AR 量子智慧戰情室 V5.0 是一款專為企業設計的應收帳款智慧管理系統，結合了傳統帳齡分析與先進的預測模型，協助財務與業務團隊做出更精準的信用決策。

### 1.2 核心功能

- **全方位帳款監控**：即時掌握應收帳款狀況
- **客戶生命週期分析**：從信用演進角度管理客戶
- **授信組合優化**：將授信視為投資組合進行配置
- **預測引擎**：提前預測未來帳款變化
- **業務歸因**：將帳款責任歸因到業務人員

### 1.3 系統需求

- 瀏覽器：Chrome / Edge / Firefox（最新版本）
- Excel 檔案：.xlsx 格式（.xls 需另存為 .xlsx）

## 3. 資料上傳

### 3.1 支援的資料格式

系統支援 Excel (.xlsx) 格式的資料上傳。請確保資料包含以下欄位：

**必要欄位**：
- 客戶名稱
- 帳款金額
- 帳齡天數
- 交易日期

**選填欄位**（影響分析深度）：
- 客戶營業額
- 業務員名稱
- 風險等級
- 信用評分

### 3.2 上傳流程

1. 點擊頁面左上角的「📁 上傳資料」按鈕
2. 選擇您的 Excel 檔案
3. 系統自動解析並更新所有圖表
4. 右上角顯示「資料更新成功」提示

### 3.3 資料驗證

上傳後系統會檢查：
- 必要欄位是否齊全
- 數值格式是否正確
- 是否有異常值（如負數金額）

如發現問題，會顯示紅色警示訊息說明原因。

## 4. 各 Tab 操作說明

### 4.1 Tab 1：儀表板總覽

**呈現內容**：
- 總應收帳款金額
- 平均帳齡
- 壞帳率
- 系統健康度分數

**操作方式**：
- 預設顯示當月資料
- 點擊 KPI 卡片可查看趨勢圖

### 4.2 Tab 2：帳齡分析

**呈現內容**：
- 帳齡分布長條圖（0-30 / 31-60 / 61-90 / 91-120 / 120+ 天）
- 帳齡趨勢折線圖
- 各帳齡區間的金額與佔比

**操作方式**：
- 滑鼠移到長條上顯示詳細金額
- 點擊圖例可隱藏/顯示特定區間

### 4.3 Tab 3：客戶評級

**呈現內容**：
- 客戶評分排行榜
- 營收貢獻度分析
- 風險分佈矩陣

**操作方式**：
- 排序切換：按金額 / 按評分 / 按風險
- 搜尋特定客戶名稱

### 4.4 Tab 4：催收管理

**呈現內容**：
- 催收案件清單
- 催收成效統計
- 建議催收優先順序

**操作方式**：
- 勾選催收完成狀態
- 輸入催收備註

### 4.5 Tab 5：客戶生命週期 ⭐ V5 新功能

**呈現內容**：
- 四象限散點圖：客户發展階段分布
- 信用成熟度演進圖：歷史評分變化
- 策略矩陣：對應的授信策略

**操作方式**：
- 散點圖：點擊客戶查看詳情
- 演進圖：勾選要比較的客戶（最多5個）
- 矩陣：hover 顯示策略說明

**解讀指南**：

| 象限 | 客戶階段 | 建議策略 |
|------|---------|---------|
| 左上 | 新客戶期 | 觀察為主，審慎授信 |
| 右上 | 成長期 | 適時提高額度 |
| 右下 | 成熟期 | 穩定合作，優化條件 |
| 左下 | 衰退期 | 收緊額度，加強催收 |

### 4.6 Tab 6：授信組合矩陣 ⭐ V5 新功能

**呈現內容**：
- 2×2 組合矩陣：業績 × 風險
- 目前 vs. 建議配置甜甜圈圖
- 最佳化收益預測

**操作方式**：
- 點擊矩陣格子查看該類別客戶清單
- 甜甜圈圖切換：金額 / 客戶數

**解讀指南**：

| 類型 | 定義 | 策略 |
|------|------|------|
| 明星 | 高業績 + 低風險 | 主動維護，提高額度 |
| 潛力 | 低業績 + 低風險 | 開發需求，提高業績 |
| 穩定 | 低業績 + 高風險 | 觀察為主 |
| 問題 | 高業績 + 高風險 | 優先催收，檢視風險 |

### 4.7 Tab 7：預測引擎 ⭐ V5 新功能

**呈現內容**：
- 30/60/90 天帳齡趨勢預測
- 還款機率熱力圖
- 三情境分析

**操作方式**：

**What-If 模擬**：
1. 拖動「年化利率」滑桿（0-15%）
2. 拖動「催收強度」滑桿（0.5x-2.0x）
3. 選擇「模擬期間」（1-6 個月）
4. 圖表即時更新預測結果

**情境切換**：
- 點擊「樂觀」/「基準」/「悲觀」切換情境
- 預測區段以虛線顯示

**⚠️ 注意**：預測功能需至少 6 個月歷史資料才能產生準確預測。

### 4.8 Tab 8：業務歸因 ⭐ V5 新功能

**呈現內容**：
- 業務員帳齡績效排行
- 信用管控雷達圖
- 業務 × 風險熱力矩陣

**操作方式**：
- 排行榜：切換排序維度
- 雷達圖：勾選要比較的業務員（最多3人）
- 矩陣：點擊查看該格詳情

**應用場景**：
- 業務檢視：我管理的客戶帳款好嗎？
- 管理審視：哪個業務需要協助？
- 績效考核：帳款回收作為獎金參考

## 5. 篩選與參數設定

### 5.1 時間範圍篩選

位置：篩選列左側

選項：
- 近 3 個月
- 近 6 個月（預設）
- 近 12 個月
- 自訂範圍

### 5.2 客戶與業務員篩選

位置：篩選列

- 下拉選單可選擇特定客戶或業務員
- 多選功能（按住 Ctrl/Cmd）
- 清除篩選：點擊「✕ 清除」

### 5.3 參數調整面板

位置：工具列「⚙ 參數調整」

可調整項目：

| 參數 | 說明 | 預設值 |
|------|------|-------|
| 帳齡閾值 | 各級帳齡天數 | 30/60/90/120 |
| 風險權重 | 風險因素計算權重 | 標準 |
| 預測信心度 | 預測區間寬窄 | 95% |
| 警示閾值 | 觸發警示的條件 | 標準 |

調整後點擊「✅ 套用」立即生效。

## 6. 匯出與分享

### 6.1 匯出功能

位置：工具列「📥 匯出」

支援格式：
- **PDF**：完整報告（所有 Tab）
- **Excel**：原始資料表
- **PNG**：當前檢視的圖表

### 6.2 匯出設定

- 選擇要匯出的 Tab
- 設定頁面大小（A4 / Letter）
- 包含篩選條件：是/否
- 匯出按鈕點擊後下載開始

### 6.3 分享功能

- 點擊「🔗 分享」產生連結
- 連結包含目前的篩選狀態
- 連結有效期：7 天

## 7. 常見問題

### Q1：上傳資料後圖表沒有反應？

**解決方案**：
1. 檢查瀏覽器是否封鎖彈出視窗
2. 確認 Excel 欄位名稱正確
3. 嘗試重新整理頁面（F5）
4. 檢查瀏覽器 Console 有無錯誤訊息

### Q2：預測數據顯示「資料不足」？

**原因**：歷史資料少於 6 個月

**解決方案**：
- 累積更多歷史資料後再使用預測功能
- 或聯繫管理員匯入歷史資料

### Q3：業務歸因看不到我的資料？

**原因**：資料中沒有「業務員」欄位

**解決方案**：
- 確認 Excel 資料包含業務員欄位
- 或請管理員協助對應業務資料

### Q4：系統健康度是怎麼計算的？

**公式**：
```
健康度 = 資料完整性(30%) + 帳齡健康度(30%) + 催收效率(20%) + 預測準確度(20%)
```

- 綠色（>80%）：正常
- 黃色（60-80%）：需要注意
- 紅色（<60%）：需要關注

### Q5：如何回存設定？

點擊「💾 儲存設定」可儲存：
- 目前的篩選條件
- 參數面板設定
- 偏好的 Tab

下次開啟會自動恢復。
